# 📚 RAG Quiz Generator

## 📌 Deskripsi
Aplikasi ini menggunakan **RAG (Retrieval-Augmented Generation)** dengan model **Llama-3-8B-Instruct** untuk menghasilkan soal ujian otomatis dari materi PDF.Dengan mengadopsi model **Llama-3-8B-Instruct** dari Hugging Face dan didukung oleh **HuggingFace Embeddings (BAAI/bge-m3)**, aplikasi ini mampu menganalisis konten PDF yang diunggah dan menghasilkan soal-soal relevan dengan berbagai tingkat kesulitan dan format.

## 🚀 Fitur Utama
- Upload PDF materi kuliah
- Generate soal dengan 3 tipe: Pilihan Ganda, Isian Singkat, Benar/Salah
- Tingkat kesulitan: LOTS (Hafalan) atau HOTS (Analisis)
- Export soal ke TXT, JSON, Excel, dan PDF

## 🏗️ Teknologi yang Digunakan
- **LLM:** Meta-Llama-3-8B-Instruct (HuggingFace)
- **RAG:** LangChain + FAISS
- **Embedding:** HuggingFace Embeddings (BAAI/bge-m3)
- **UI:** Streamlit
